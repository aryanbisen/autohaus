# SecureDrive

Willkommen bei **SecureDrive**, einer kompromisslos sicheren Plattform für den Kauf und Verkauf von Fahrzeugen. Da beim Autohandel hohe Summen, sensible persönliche Daten und Vertrauen im Spiel sind, wurde dieses Projekt nach dem Prinzip **"Security by Design"** entwickelt. Der Fokus liegt auf der strikten Einhaltung der **OWASP Top 10** Sicherheitsrichtlinien, um Betrug, Datenlecks und Cyberangriffe effektiv zu verhindern.
